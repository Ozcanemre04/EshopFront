# EshopFront
  An e-commerce web app that consumes EshopApi

## Production
  https://eshop-front-eight.vercel.app

## Prerequisites
 - Node.js
 - Angular
 - Git

## Installation
 
1. Clone repository

   git clone https://github.com/Ozcanemre04/EshopFront.git

2. Navigate to project

   cd EshopFront   

3. Install dependency

   npm install

4. Build the project

   ng build --configuration development

5. Run the project

   ng serve
